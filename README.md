# 📉 Customer Churn Analysis & Prediction

 📌 Objective
The goal of this project is to analyze customer behavior and identify factors that lead to customer churn. A machine learning model is built to predict whether a customer is likely to leave.

 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Machine Learning (Scikit-learn)

 📂 Dataset
- Telecom Customer Churn Dataset (Kaggle)
- Contains customer details such as tenure, monthly charges, contract type, and churn status

 🔍 Key Analysis Performed
- Data Cleaning (handling missing values, data type conversion)
- Exploratory Data Analysis (EDA)
- Churn analysis based on:
  - Tenure
  - Monthly Charges
  - Contract Type

 📊 Key Insights
- Customers with **low tenure** are more likely to churn
- Higher **monthly charges** increase the probability of churn
- Customers with **month-to-month contracts** have the highest churn rate
- Long-term customers are more stable

 🤖 Machine Learning Model
- Model Used: Logistic Regression
- Accuracy: 80%   

💡 Business Recommendations
- Offer incentives for long-term contracts
- Focus on retaining new customers
- Optimize pricing strategies for high-risk customers
- Improve onboarding experience
